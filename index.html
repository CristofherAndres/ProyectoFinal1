Primeros pasos con git

Este es un proyecto de prueba para aprender a usar git.

Iniciarlizar un repositorio

Para iniciar un repositorio git en un proyecto, debemos ejecutar el comando git init en la raíz del proyecto.

Configurar git

Para configurar git con nuestro nombre y correo, debemos ejecutar los siguientes comandos:

        
            git config --global user.name "Tu nombre"
            git config --global user.email "tu correo"
